# Anti-Overlap Checklist

Use this checklist before launch and after each major deployment change.

## Required checks
- binary names
- config names
- datadirs
- chain magic
- genesis hash
- ports
- service names
- pool ports
- explorer database
- wallet database
- RPC cookie path
- website branding
- explorer branding
- wallet branding
- pool branding
- API route names
- CSS titles/logos
- HBYC text
- MBUX text
- BTC/BCH/DGB accidental labels

## Pass criteria
- Every check above must be verified as HOBC clean-room compliant.
- Any overlap with old HOBC, MBUX, BTC, BCH, or DGB naming/config must be fixed before release.
