HOBC Cold Storage

Generate, restore, receive, and sign fully offline.

HobbyHash Cold Wallet is built for an offline Windows PC. Offline cold storage wallet. This app does not connect to the internet, RPC, explorer, or mining pool.

Offline-use warning: Keep this app on an air-gapped computer. Never type your recovery phrase into a website, chat, support form, hosted wallet, or online computer.

Local seed generation

Creates a 24-word BIP39 recovery phrase with secure random bytes on this computer only.

HOBC receive addresses

Derives native segwit addresses using the HobbyHash Coin network HRP and prefixes.

Offline signing

Imports unsigned transaction JSON or PSBT-style files from USB and exports signed files only.
